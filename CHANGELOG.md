## [1.0.0] - 2026-03-19

### Added
- 22 deterministic skills across 5 categories (core, syntax, impl, errors, agents)
- Deep research via WebFetch on all official Vite docs (vite.dev)
- Comprehensive vooronderzoek with 3 research fragments
- Definitive masterplan with 8 execution batches and full agent prompts
- INDEX.md with complete skill catalog
- README.md with installation instructions, version coverage, quality guarantees

### Skills Created
- **Core (2)**: vite-core-architecture, vite-core-environment-api
- **Syntax (8)**: vite-syntax-config, vite-syntax-server, vite-syntax-build, vite-syntax-resolve-css, vite-syntax-plugin-api, vite-syntax-hmr-api, vite-syntax-assets, vite-syntax-env-vars
- **Implementation (7)**: vite-impl-project-setup, vite-impl-library-mode, vite-impl-ssr, vite-impl-backend-integration, vite-impl-optimization, vite-impl-javascript-api, vite-impl-migration
- **Errors (3)**: vite-errors-build, vite-errors-dependency, vite-errors-dev-server
- **Agents (2)**: vite-agents-review, vite-agents-project-scaffolder

### Key Decisions
- D-008: Version scope updated to Vite 6, 7, and 8 (v5 migration only)
- D-010: Domain updated from vitejs.dev to vite.dev
- D-011: 22 skills finalized across 8 batches
- D-012: WebFetch verification mandatory for all content

### Research Findings
- Vite 8 uses Rolldown (not Rollup) and Oxc (not esbuild)
- Official domain migrated from vitejs.dev to vite.dev
- Environment API introduced in Vite 6 (RC phase)
